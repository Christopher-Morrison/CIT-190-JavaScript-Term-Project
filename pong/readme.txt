# Pong Game


## Description

The goal of this game is to bounce the ball between the two paddles on each side.
Missing the ball will result in a loss. 

There are up to two players. Player one will use 'W' and 'S' to control their side. 
Player two will use the up and down arrow keys.


## Usage

You can change the difficulty by moving the difficulty slider. This will speed up the game. 
Once you are ready to play, click play game. Once finished, click reset to reset the game.
